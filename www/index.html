About

NVC is a free software VHDL compiler and simulator implementing almost all of IEEE 1076-2002.

Find out more in the README or read the manual online.

The project is hosted on both GitHub and SourceHut.

The most recent version is 1.5.3, released on 13th November 2021 (release notes).
