Download

The most recent version is 1.5.3, released on 13th November 2021 (release notes).

nvc-1.5.3.tar.gz

Releases are signed using PGP key BCDB295F74319F1A. Download nvc-1.5.3.tar.gz.sig and verify with: 

gpg --keyserver pgp.mit.edu --recv-keys BCDB295F74319F1A
gpg --verify nvc-1.5.3.tar.gz

Refer to the README for instructions on how to build and install the software.

Development version

The latest source code can be fetched using Git: 

git clone https://git.nickg.me.uk/nvc.git

Alternatively there are mirrors on GitHub and SourceHut.
